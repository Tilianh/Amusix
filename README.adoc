== What is Amusix?

Amusix in an application powered by the YouTube API to listen to and share music playlists with your friends.

It is based on .NET 10 and Angular 21.

== Code structure

* `AmusixBackapp`: backend application
** Based on .NET 10
* `AmusixFrontapp`: frontend application
** Based on Node 24 and Angular 21
* `AmusixTestapp`: test application
** Based on Node 24 and Cypress 15
