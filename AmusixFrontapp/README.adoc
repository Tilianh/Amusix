== Stack

=== Main stack

* https://nodejs.org/fr[Node.js 24]
* https://v21.angular.dev/overview[Angular 21] (+ TypeScript)
* https://getbootstrap.com/docs/5.3[Bootstrap 5.3] (+ SCSS)

* https://www.npmjs.com/package/universal-cookie/v/8.1.2[`universal-cookie` (8.1.2)]: browser cookie management
* https://www.npmjs.com/package/youtube-player/v/5.6.0[`youtube-player` (5.6.0)]: YouTube player integration and interactions

== File structure

=== Project structure

* `src/`:
** `app/`:
*** `core/`: core components
*** `features/`:
**** `auth/`: authentication resources
***** Login form
***** Register form
**** `home-page/`: application home-page
**** `playlists/`: playlist resources
***** Modal to add song to a playlist
***** Current user's playlists page
***** Playlist consultation / song listing page
***** Playlist editing modal
***** Playlist listing item
***** Playlist selection item
***** Playlist song listing item
**** `songs/`: songs resources
***** Search result song listing item
***** Song player
***** Song search bar
***** Song search results page
**** `user-profile/`: user profile resources
***** Current user's profile page
***** Password changing modal
*** `shared/`: shared / reusable resources
*** `app.routes.ts`: page routes
** `environment/`: environment files
** `styles/`: styling sheets

=== Resource directories' structure

* `components/`: Angular component files (HTML, SCSS and TS)
* `models/`: data models (user, playlist, song, etc.)
* `services/`: services (component communication, API calls, etc.)

== Setup

=== Dependency installation

. Install Node.js
. Install all project dependencies by running the following command at the frontend project's root in a terminal:
+
[source, bash]
----
npm i
----

=== Environment files

. Add / edit the environment files in `src/environments`:
+
* `environment.development.ts`: local development environment file
* `environment.production.ts`: production environment file

. Add / replace the following fields in the files:
+
* `apiUrl`: API URL (ex: `http://localhost:5211`)

== Launch

Run the following command at the frontend project's root in a terminal to launch the application in a local development environment:

[source, bash]
----
ng serve
----

Default access URL: http://localhost:4200

== Build

Run the following command at the frontend project's root in a terminal to build the application for a production environment:

[source, bash]
----
ng build --base-href <base-href>
----

Replace `<base-href>` with the corresponding deployment base URL -> If the production deployment base URL is https://amusix.xyz, `<base-href>` must be replaced with `https://amusix.xyz/`

WARNING: The `/` at the end of the specified base URL must not be forgotten!

A default `.htaccess` file is provided in the build to manage request redirections

Default build output directory: `/dist` (at frontend project's root)
